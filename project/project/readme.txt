頁面設計說明
#首頁
    -遊戲開始按鈕

#主畫面
    -對話介面
    -側欄項目：
    主線劇情、動物圖鑑、探索地圖、研究實驗室、成就系統、物品背包

#側欄延伸介面
    共6頁
    -主線劇情：遊戲發展
    -動物圖鑑：回答選擇題的問題，回答正確即可獲得該圖鑑
    -探索地圖：點擊查看解鎖及尚未解鎖的區域
    -研究實驗室：玩家可裝飾小屋或養成動物
    -成就系統：設立目標分數或目標項目(例如：收集5隻動物)
    -物品背包：存放收集到的道具

#遊戲1
    -連連看
    -遊戲規則：動物emoji連線至動物的中文名

#遊戲2
    -找不同
    -遊戲規則：找兩張圖片不圖之處

#遊戲3
    -迷宮
    -遊戲規則：使用鍵盤或按介面上的上下左右鍵走出迷宮
